Mesh Embedding Module

Header File

Mesh_Embedding.h

Function Purpose

Implements Tutte's embedding algorithm to parameterize 3D open meshes with annotated boundaries onto a 2D triangular domain.

Function Signature

auto embedding_result = MeshEmbedding::TutteEmbedding::Parameterize(

    const std::vector<std::vector<size_t>>& boundary_chains,
    const size_t weight_type = 1,
    const size_t check_interval = 100,
    const size_t max_iterations = 100000000
);
